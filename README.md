# Repositório de AEDS I - Algoritmos e Estruturas de Dados

Bem-vindo ao meu repositório de AEDS I Ciência da Computação PUC Minas! Este repositório contém implementações em C dos conceitos fundamentais da disciplina de Algoritmos e Estruturas de Dados I. 
O objetivo é organizar e compartilhar meu aprendizado, além de servir como um recurso para outros estudantes.

## 📚 Conteúdos Abordados

### ✅ Conceitos Básicos
- Introdução à linguagem C e estrutura básica de um programa
- Variáveis, tipos de dados e operadores
- Entrada e saída padrão

### ✅ Estruturas Condicionais e Repetição
- Estruturas de decisão: `if`, `else`, `switch`
- Estruturas de repetição: `for`, `while`, `do while`

### ✅ Vetores e Matrizes
- Declaração e manipulação de vetores e matrizes
- Operações básicas: inserção, remoção e busca

### ✅ Strings
- Manipulação de strings em C
- Funções úteis da biblioteca string.h

### ✅ Arquivos
- Leitura e escrita de arquivos
- Operações básicas com arquivos texto

### ✅ Ponteiros
- Conceito de ponteiros e alocação dinâmica de memória
- Uso de ponteiros em funções

### 🔄 Programação Orientada a Objetos (POO) em C++
- Introdução aos conceitos de POO, como classes e objetos
- Simulação de POO em C++ com exemplos práticos

### 🔄 Recursão
- Conceito de recursão e exemplos clássicos
- Comparação entre recursão e iteração

## 📂 Estrutura do Repositório

conceitos-basicos/ → primeiros programas em C, compilação e execução.

estruturas-condicionais/ → if, else, switch.

estruturas-repeticao/ → while, do-while, for.

vetores/ → introdução a arrays.

matrizes/ → operações com matrizes.

strings/ → manipulação de cadeias de caracteres.

arquivos/ → leitura e escrita em arquivos.

ponteiros/ → conceitos e aplicações.

recursao/ → funções recursivas.

orientacao-objetos/ → introdução ao paradigma OO em C+.

## 🛠️ Como Usar Este Repositório

1. **Navegue até o arquivo desejado**: Clique no arquivo que você gostaria de abrir.
2. **Baixe o arquivo para o seu computador**: Use a opção de download para salvar o arquivo localmente.
3. **Compile o código**: Utilize um compilador de sua preferência para compilar e executar o programa.
